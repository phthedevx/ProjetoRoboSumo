# 🤖 Projeto Integrador - Robô de Sumô

## 🧠 Sobre o Projeto
Este repositório contém o desenvolvimento do nosso **Robô de Sumô**, criado como parte do Projeto Integrador entre as turmas de **Internet das Coisas (IoT)** e **Eletrônica Aplicada**.

Nosso grupo optou pela modalidade **Sumô de Robôs**, cujo objetivo é construir um robô capaz de detectar, perseguir e empurrar o robô adversário para fora da arena circular.

- 1 Ponte H L293D

## 🎯 Objetivos do Projeto
- Estimular o trabalho em equipe entre as turmas de IoT e Eletrônica Aplicada.  
- Aplicar conhecimentos práticos de programação e eletrônica embarcada.  
- Desenvolver um robô autônomo competitivo e eficiente.  
- Participar de uma competição saudável e técnica.
